# Purpose

This is a sample setup to demonstrate how to customize the [Worklists plugin](https://orthanc.uclouvain.be/book/plugins/worklists-plugin-new.html).

# Description

This demo contains:

- an Orthanc container that is running OE2 and the Worklists plugin.  Check the `ORTHANC_JSON` configuration in the `docker-compose.yml` file.


# Starting the setup

To start the setup, type: `docker-compose up`

# demo

- Orthanc UI is accessible at [http://localhost:8042/ui/app/](http://localhost:8042/ui/app/)
