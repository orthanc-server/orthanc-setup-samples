# Purpose

This is a sample setup to demonstrate how to configure the Education plugin and the DICOMizer.

# Description

This demo contains:

- an Orthanc container (configured to display the [Education plugin](https://orthanc.uclouvain.be/book/plugins/education.html))

# Starting the setup

To start the setup, type: `docker compose up -d`

# Demo
- Reach `http://localhost:8042` in your favorite web browser
- Fill the credentials (here: `admin` and `admin`)
- Upload some whole-slide images thanks to the `DICOM-ization of images` page (left panel)
